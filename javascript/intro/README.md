# Intro to JavaScript

## Objectives

* 5 Primitive Datatypes in JavaScript

```javascript
//Numbers
1 2 3 4

//Modulo - remainder operator

10 % 3   //1

----------------------
//Strings
"Hello World"

//Single or Double quotes OK
"hello Erik"
'hello Erik'

//Concatenation
"Erik" + "P"  //"ErikP"


//Strings have a length property
"hello Erik".length  //10

//Access individual characters using [] and an index
"Erik"[0]  //"E"

----------------------

//Booleans
true
false

----------------------

//null and undefined
null
//Variables that are declared but not
//initialized are undefined
//The following variables are undefined:
var name;

undefined
//null is "explicitly nothing"
var currentPlayer = "E";
currentPlayer = null;   //game over

-----------------------

//Variables

//Variables are simply containers that store values
//They follow this pattern:
var yourVariableName = yourValue;

//They can store all of the values we've seen
var name = "Erik";

//Recall the stored value by calling the variable name
var name = "Erik";
"hello there " + name    //"hello there Erik"

var num = 12;
num + 2 + 10    //24

//We can also update existing variables

```
